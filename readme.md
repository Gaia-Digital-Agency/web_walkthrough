# Customer Journey X-Ray Mockup

A 3-page educational website that explains the 12 psychological areas of a customer walkthrough.

## Installation
1. Create a folder named `journey-mockup`.
2. Copy the code blocks for `index.html`, `product.html`, and `contact.html`.
3. Create a `style.css` file (copy the SCSS code providedâ€”it is compatible with standard CSS as written).
4. Create a `main.js` file (copy the TypeScript code provided).
5. Open `index.html` in any browser.

## How to Use
- **Standard View:** The site looks like a normal business landing page.
- **X-Ray View:** Click the "Toggle Journey X-Ray" button in the bottom right.
- **Education:** Observe the color-coded boxes and labels (1-12) to see how the website sections map to user psychology.
