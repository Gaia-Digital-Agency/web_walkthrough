# Customer Journey X-Ray Mockup

A 3-page educational website that explains the 12 psychological areas of a customer walkthrough.

Original source: https://github.com/Gaia-Digital-Agency/web_walkthrough

## Installation

```bash
npm install
npm run dev
```

## How to Use

- **Standard View:** The site looks like a normal business landing page.
- **X-Ray View:** Click the "Toggle Journey X-Ray" button in the bottom right.
- **Education:** Observe the color-coded boxes and labels (1-12) to see how the website sections map to user psychology.

## Pages

- `index.html` - Home page with hero, trust bar, and problem statement
- `pages/product.html` - Product features, security, and lead generation
- `pages/contact.html` - Contact form with success state

## The 12 Psychological Steps

| Step | Name | Phase |
|------|------|-------|
| 1 | Pre-Arrival Priming | Awareness |
| 2 | Technical Greeting | Awareness |
| 3 | Hero Orientation | Awareness |
| 4 | Navigation Exploration | Awareness |
| 5 | Narrative Scroll | Logic |
| 6 | Evidence & Social Validation | Trust |
| 7 | Education & Deep Dive | Logic |
| 8 | Risk Mitigation | Trust |
| 9 | Micro-Conversion | Action |
| 10 | Intent Expression | Action |
| 11 | Frictionless Input | Action |
| 12 | Post-Conversion | Action |

## Color Coding

- **Blue** (#3b82f6) - Awareness phase
- **Purple** (#8b5cf6) - Logic phase
- **Green** (#10b981) - Trust phase
- **Orange** (#f59e0b) - Action phase
